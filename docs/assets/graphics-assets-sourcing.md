# Graphics Asset Sourcing Notes

## Purpose
This document defines how original Desert Rats visual files are collected, verified, and tracked.

## Collection Rules
- Use only original Desert Rats files from trusted archives.
- Keep direct source URLs and local filenames synchronized with the register.
- Preserve original format whenever available (GIF/JPG/SCR).

## Quality Grading
- A: Clear, high-resolution, directly useful reference.
- B: Useful but partial/low-res.
- C: Weak reference, use only if no better source exists.

## IP Risk Levels
- low: source archive trusted and project has permission from rights holder.
- medium: source is trusted but legal provenance is incomplete.
- high: source is unverified or ownership unclear.

## Required Metadata Per Entry
- asset_id
- source_url
- what_it_documents
- quality_grade
- ip_risk
- recommended_use
- local_file

# Engine Specification Package v1

This package defines the normative engine mechanics specification for the currently implemented runtime behavior.

## Audience
- Engine developers
- QA and test authors
- Scenario tooling and runtime integration maintainers

## Scope Rule
This package is normative only for mechanics that are implemented in code and validated by tests.

Out-of-scope mechanics are listed explicitly as non-normative backlog items.

## Documents
- `engine-spec-v1.md` - master normative specification and glossary.
- `turn-semantics-v1.md` - formal one-turn transition contract.
- `scenario-runtime-contract-v1.md` - scenario loading and validation contract.
- `determinism-contract-v1.md` - reproducibility and semantic equivalence contract.
- `traceability-matrix-v1.md` - requirement and evidence matrix.

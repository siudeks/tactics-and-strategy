# Windows Release to itch.io

This runbook describes how to produce and upload a portable Windows app image with native EXE launcher and bundled Java runtime.

## Prerequisites
- GitHub repository secrets:
  - `ITCH_IO_API_KEY`
  - `ITCH_IO_GAME_ID` (format: `username/game-slug`)
- Access to workflow dispatch on GitHub Actions

## Local Build (Windows only)
From repository root:

```bash
./gradlew lwjgl3:createWindowsPortableAppImage
```

Generated directory:
- `lwjgl3/build/distributions/portable/tactics-and-strategy/`

Artifact type:
- Portable app-image generated by `jpackage` with native `tactics-and-strategy.exe` launcher.

## Publish Through GitHub Actions
1. Open Actions and run workflow `Release Windows Build to itch.io`.
2. Set `channel` (for example `windows`).
3. Set `version` (for example `0.1.0-test1`).
4. Start workflow.

The workflow runs on `windows-latest`, builds portable app-image, unpublishes the previous channel content, and uploads the app-image directory using butler.

## Tester Verification
1. Tester installs or updates build via itch app channel.
2. Tester runs `tactics-and-strategy.exe` from installed portable folder.
3. Game starts without any system Java installation.
